* SpringDataJpa
* Hibernate
* PostgreSQL
* Projeto Lombok
* Swagger
* DBeaver
* Docker


### Executando a aplicação

Para executar a aplicação é simples, basta abrir a classe `TasksApiApplication` e ativar `run` em sua IDE.

ou executar o comando abaixo:

```shell
mvn spring-boot:run
```
